\textbf{20.} Estudia la continuïtat de la funció:
\[
f(x) = \begin{cases} 
2x + 1 & \text{si } x \le -1 \\ 
\frac{x + 3}{2x} & \text{si } -1 < x \le 1 \\ 
\frac{x}{x - 2} & \text{si } x > 1 
\end{cases}
\]

\vspace{0.5cm}
\textbf{1. Punts susceptibles a no ser contínua:}
\vspace{0.2cm}

Hem d'estudiar dos tipus de punts: els punts de ruptura (on canvia la definició de la funció) i els punts on els denominadors s'anul·len (condicions de no domini).

\begin{itemize}
    \item \textbf{Punts de ruptura:} tenim canvis de tram en $x = -1$ i $x = 1$.
    \item \textbf{Punts de no domini per a cada tram:}
    \begin{itemize}
        \item Al segon tram, $\frac{x + 3}{2x}$, el denominador s'anul·la si $2x = 0 \Rightarrow x = 0$. Com que el punt $x = 0$ pertany a l'interval d'aquest tram $(-1 < 0 \le 1)$, és un punt d'estudi.
        \item Al tercer tram, $\frac{x}{x - 2}$, el denominador s'anul·la si $x - 2 = 0 \Rightarrow x = 2$. Com que el punt $x = 2$ pertany a l'interval d'aquest tram $(2 > 1)$, també és un punt d'estudi.
    \end{itemize}
\end{itemize}

En resum, cal estudiar la continuïtat en els punts: \textbf{$x = -1$, $x = 0$, $x = 1$ i $x = 2$}.

\vspace{0.5cm}
\textbf{2. Anàlisi individual dels punts:}
\vspace{0.2cm}

\textbf{Estudi en $\mathbf{x = -1}$ (punt de ruptura):}
\begin{itemize}
    \item \textbf{Imatge:} $f(-1) = 2(-1) + 1 = -1$
    \item \textbf{Límits laterals:}
    \[ \lim_{x \to -1^-} f(x) = \lim_{x \to -1^-} (2x + 1) = -1 \]
    \[ \lim_{x \to -1^+} f(x) = \lim_{x \to -1^+} \frac{x + 3}{2x} = \frac{-1 + 3}{2(-1)} = \frac{2}{-2} = -1 \]
\end{itemize}
Com que la imatge coincideix amb els límits laterals ($f(-1) = \lim_{x \to -1} f(x) = -1$), la funció \textbf{és contínua} en $x = -1$.

\vspace{0.5cm}
\textbf{Estudi en $\mathbf{x = 0}$ (punt de no domini):}
\begin{itemize}
    \item \textbf{Imatge:} $f(0) = \frac{0 + 3}{2(0)} = \frac{3}{0} \Rightarrow \text{No existeix.}$
    \item \textbf{Límit:} Al calcular el límit obtenim $k/0$, per tant, calculem els límits laterals:
    \[ \lim_{x \to 0^-} \frac{x + 3}{2x} = \frac{3}{0^-} = -\infty \]
    \[ \lim_{x \to 0^+} \frac{x + 3}{2x} = \frac{3}{0^+} = +\infty \]
\end{itemize}
La funció presenta una \textbf{discontinuïtat de salt infinit} (asimptòtica) en $x = 0$.

\vspace{0.5cm}
\textbf{Estudi en $\mathbf{x = 1}$ (punt de ruptura):}
\begin{itemize}
    \item \textbf{Imatge:} $f(1) = \frac{1 + 3}{2(1)} = \frac{4}{2} = 2$
    \item \textbf{Límits laterals:}
    \[ \lim_{x \to 1^-} f(x) = \lim_{x \to 1^-} \frac{x + 3}{2x} = 2 \]
    \[ \lim_{x \to 1^+} f(x) = \lim_{x \to 1^+} \frac{x}{x - 2} = \frac{1}{1 - 2} = \frac{1}{-1} = -1 \]
\end{itemize}
Com que els límits laterals són finits però diferents ($2 \neq -1$), la funció presenta una \textbf{discontinuïtat de salt finit} en $x = 1$.

\vspace{0.5cm}
\textbf{Estudi en $\mathbf{x = 2}$ (punt de no domini):}
\begin{itemize}
    \item \textbf{Imatge:} $f(2) = \frac{2}{2 - 2} = \frac{2}{0} \Rightarrow \text{No existeix.}$
    \item \textbf{Límit:} Novament, obtenim $k/0$, per tant, calculem els límits laterals:
    \[ \lim_{x \to 2^-} \frac{x}{x - 2} = \frac{2}{0^-} = -\infty \]
    \[ \lim_{x \to 2^+} \frac{x}{x - 2} = \frac{2}{0^+} = +\infty \]
\end{itemize}
La funció presenta una \textbf{discontinuïtat de salt infinit} (asimptòtica) en $x = 2$.

\vspace{0.5cm}
\textbf{Conclusió:}
La funció $f(x)$ és contínua en tot el seu domini, és a dir, en $\mathbb{R} \setminus \{0, 1, 2\}$.
